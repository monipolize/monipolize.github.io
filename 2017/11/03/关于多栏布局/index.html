关于多栏布局和自适应问题

今天我们来谈谈多栏布局吧！Let’s Go 🚄🚄🚄🚄🚄

下面是我日常遇见过得双栏布局和三栏布局，当然还有其他布局模式，BUT今天就先说说这两个吧

①、关于两栏布局：
两栏布局顾名思义就是主内容分为左右两侧各有一栏。实现两栏布局有很多种办法在这里，我将最常见的几种方法列举出来

方法一：最简单粗暴的方式，百分比这里我就不写出来了，相信大家都会
方法二：BFC方式

<style>
     #header, #footer{ 
        height: 100px;
        background: red;
     }
     #content .right{
         height: 200px;
         background: green;
         overflow: auto;
     }
     #content .left{
         height: 200px;
         width: 100px;
         background: blue;
         float: left;
     }
 </style>





right right right right right right



原理：将左侧区域浮动，再讲右侧区域设置auto，形成BFC独立区域，块级上下文

方法三：负边距大法

<style type="text/css">
     html,body{
         padding: 0;
         margin: 0;
     }
     #header, #footer{ 
         height: 100px;
         background: red;
        overflow: hidden;
      }
     #main{
        overflow: auto;
     }
     #main .center{
         height: 200px;
        width: 100%;
         float: left;
     }
    .center .content{
        height: 200px;
        background: green;
        margin-right: 100px;
    }
     #main .aside{
         height: 200px;
         width: 100px;
         background: blue;
         float: left;
        margin-left: -100px;
     }
 </style>

当需要自适应的时候

#left {
    float: left;
    background-color: red;
}
#right {
    overflow: hidden;
    background-color: blue;
}

或者
.left{

// margin-left: -200px;

width:200px;
background-color:blue;
}
.right{
padding-left:200px;
background-color:red;
}
再或者
.left-3{
position:relative;
padding-right: 200px;//留出右侧内边距
height:300px;
background-color:yellow;
}
.right-3{
position:absolute;//绝对定位到右边
right:0;
height:400px;
background-color:#808080;
}

②、关于三栏布局

三栏布局是在两栏布局的基础上进行的扩展

方法一：绝对定位法

<style type="text/css">
    html，body{
        margin:0;
        height: 100%;
    }
    #left，#right{
        position: absolute;
        top: 0;
        width: 200px;
        height: 100px;
    }
    #left{
        left: 0;
    }
    #right{
        right: 0;
    }
    #main{
        margin: 0 210px;
        height: 100%;
    }
</style>
<body>
    <div id="left"></div>
    <div id="main"></div>
    <div id="right"></div>
</body>

※如果中间栏含有最小宽度限制，或是含有宽度的内部元素，当浏览器宽度小到一定程度，会发生层重叠的情况

方法二：负margin法

<style type="text/css">
    html，body{
        margin:0;
        height: 100%;
    }
    #left，#right{
        width: 200px;
        height: 100px;
        float:left;
    }
    #left{
        margin-left: -100%;
    }
    #right{
        margin-right: -200px;
    }
    #main,#body{
        margin: 0 210px;
        height:100%;
    }
    #main{
        width:100%;
        height: 100%;
        float:left;
    }
</style>
<body>
    <div id="main">
        <div id="body"></div>
    </div>    
    <div id="left"></div>
    <div id="right"></div>
</body>

方法三：浮动

<style type="text/css">
    .left{
            float:left;
            width:100px;
            height: 30px;
            background: red;
        }
        .right{
            float: right;
            width:150px;
            height: 30px;
            background: green;
        }
        .center{
            background: pink;
            height: 50px;
            margin-left: 100px;
            margin-right: 150px;
        }
</style>

<body>
    <div class="left">11</div>
    <div class="right">333</div>
    <div class="center">222</div>
</body>
