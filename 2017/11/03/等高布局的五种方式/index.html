等高布局的五种方式

方法一：伪等高
这种方式，并不是真正的实现了元素等高
上代码喽

<style>
    *{
        margin: 0;
    }
    .p{
        position: relative;
    }
    .left{
        position: absolute;
        top:0;
        left: 0;
        width:100px;
        background: green
    }
    .right{
        position: absolute;
        top: 0;
        right: 0;
        width:200px;
        background: pink;
    }
    .center{
        box-sizing:border-box;
        background-clip: content-box;
        padding:0 20px;
        border-left: 100px solid green;
         border-right: 200px solid pink;
        background: red;
    }

</style>
<body>
    <div class="par">
        <div class="left">
            <p>jjj</p>
        </div>
        <div class="center">
            <p>lllll</p>
            <p>jkljlkjl</p>
            <p>jlskjldjsas</p>
        </div>
        <div class="right">
            <p>;;;;</p>
            <p>jjjjj</p>
        </div>
    </div>
</body>

方法二：伪等高

    *{
        margin:0;
    }
    .par{
        overflow: hidden;
    }

        margin-bottom: -9999px;

方法三：真等高

<style>
    body,p{margin: 0;}
    .parent{
        display: table;
        width: 100%;
        table-layout: fixed;
    }
    .left,.centerWrap,.right{
        display: table-cell;
    }
    .center{
        margin: 0 20px;
    }
</style>
<body>
<div class="parent" style="background-color: lightgrey;">
    <div class="left" style="background-color: lightblue;">
        <p>left</p>
    </div>  
    <div class="centerWrap">
        <div class="center" style="background-color: pink;">
            <p>center</p>
            <p>center</p>
        </div>         
    </div> 
    <div class="right" style="background-color: lightgreen;">
        <p>right</p>
    </div>        
</div>

结果：

方法四：真等高

//设置子元素的top:0;bottom:0
body,p{margin: 0;}
.parent{
position: relative;
height: 40px;
}
.left,.center,.right{
position: absolute;
top: 0;
bottom: 0;
}
.left{
left: 0;
width: 100px;
}
.center{
left: 120px;
right: 120px;
}
.right{
width: 100px;
right: 0;
}

方法五：flex

<!-- 　flex中的伸缩项目默认都拉伸为父元素的高度，也实现了等高效果 -->
body,p{margin: 0;}
.parent{
    display: flex;
}
.left,.center,.right{
    flex: 1;
}
.center{
    margin: 0 20px;
}
